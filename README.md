## Sobre o Projeto

Pipeline de Big Data para análise de matrículas escolares (2022-2025) utilizando Apache Spark no Google Colab.

**Problema:** A escola parceira registrava dados manualmente em planilhas, sem conseguir analisar padrões de matrícula e cancelamento.

**Solução:** Pipeline automatizada que coleta, processa e analisa os dados, gerando métricas de gestão e previsão para 2026.

## Principais resultados

| Métrica | Valor |
|---------|-------|
| Total de matrículas (2022-2025) | 314 |
| Taxa média de conversão | 56,4% |
| Previsão para 2026 | 91 matrículas |

**Insights:** Pico de matrículas em novembro e dezembro; queda na taxa de cancelamento em 2025.
